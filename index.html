__About__

__js-vim__ is an open-source project to implement __Vim__ in web-friendly JavaScript.

__Vim__ itself is an improved implementation of the __Vi__ text editor.

Bram Moolenar is the main author of __Vim__; his effort richly deserves [sponsorship](http://www.vim.org/sponsor/).

The _js-vim_ project lives at [github.com/itsjoesullivan/js-vim](https://github.com/itsjoesullivan/js-vim), and issues should be reported to [github.com/itsjoesullivan/js-vim/issues](https://github.com/itsjoesullivan/js-vim/issues).

_js-vim_ is available for use under the __MIT__ license.



================================


__Features__

Vim's power and flexibility arise from over 20 years of coordinated effort. _js-vim_ aims to lay a foundation that can support such continued investment while bringing a quorum of Vim's features to web users.

A list of features implemented in js-vim follows. It's not a complete list of js-vim features, let alone Vim's, and in many cases expanded test coverage is needed.

- Motions
	- h, j, k, l
	- [count] motion
	- /foo, ?bar
	- w, W, b, B, e, E

- Operators
	- y, c, d, ~, gu, gU

- Text objects (_i.e., daw_)
	- w, s, p, (, {, ', "

- Registers
	- "x
	- px, Px

- Marks
	- mx
	- 'x
	- `x

- Macros
	- qx{command}q
	- @x
	- @@

- Visual select
	- v ("VISUAL")
	- V ("VISUAL LINE")
	- Ctrl-v ("VISUAL BLOCK")

__Usage__: browser

js-vim exists to provide the facility of vim to the web.

	vim.edit({
		el: $("#vim") // Your container
	});


================================

__Usage__: terminal 

	# install
	npm install -g js-vim-node

	# launch js-vim
	jsvim
